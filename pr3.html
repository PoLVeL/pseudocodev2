Windows

Γενικες πληροφοριες

Τα Microsoft Windows είναι μια σειρά από λειτουργικά συστήματα για προσωπικούς υπολογιστές και διακομιστές.

H σειρά εμφανίστηκε πρώτα από την Microsoft το 1985 για να ανταγωνιστεί το καινούριο σύστημα της Apple, τον Apple Macintosh, το οποίο χρησιμοποιούσε γραφικό περιβάλλον. Tα Microsoft Windows τελικά επικράτησαν στην παγκόσμια αγορά προσωπικών υπολογιστών με ένα μερίδιο αγοράς που υπολογίζεται περίπου στο 90%. Tα Microsoft Windows είναι λογισμικό κλειστού κώδικα. Tα Windows αναπτύχθηκαν για τους IBM συμβατούς μικροϋπολογιστές οι οποίοι ήταν βασισμένοι στην αρχιτεκτονική x86 της Intel, και σήμερα σχεδόν όλες οι εκδόσεις των Windows είναι κατασκευασμένες για αυτή την πλατφόρμα.
